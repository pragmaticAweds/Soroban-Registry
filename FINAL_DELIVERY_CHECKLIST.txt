COMMAND-LINE INTERFACE
================================================================================

SINGLE CONTRACT VERIFICATION:
  soroban-registry contract verify <ADDRESS> --network testnet [--options]

  OPTIONS:
    --network <NETWORK>     Stellar network (mainnet|testnet|futurenet)
    --json                  Machine-readable JSON output
    --strict                Fail if any warnings/errors found
    --no-cache              Skip cache and fetch fresh data

BATCH VERIFICATION:
  soroban-registry contract verify "ADDR1,ADDR2,ADDR3" --batch [--options]

  Same options plus:
    --batch                 Process multiple contracts
